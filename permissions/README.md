# Shell Permissions Scripts

## 0-iam_betty
This script switches the current user to the user 'betty' using the `su` command 
[200~## 1-who_am_i

## 2-groups

## 3-new_owner
This script changes the owner of the file `hello` to the user `betty` using the `chown` command.
